# Store
  
[The Saito Store](https://saito.io/store/) is a decentralized e-commerce app that runs atop the Saito Network. It allows sellers to list items for sale, and buyers to buy them. All sales are processed directly on-chain as atomic swaps. There is no point where sellers lose control of their listings.

This makes the Saito Store different from private marketplaces like Shopify and Amazon, where users have to transfer ownership. Instead of taking possession of items, the Saito Store merely indexes the listings that sellers publish on-chain. When a user purchases an item the server tracks the transfer and updates its indexes.

### Why Build a Decentralized Shopify / Amazon?

Asynchronous on-chain commerce is a powerful feature needed for many Saito apps. Users who create NFTs should be able to sell them online. Users who publish blog posts should be able to sell access keys. We believe developers should be able to release games and applications and CSS themes and charge for them as well.

Given that the Saito blockchain is a decentralized blockchain powered by a distributed consensus algorithm, the Saito team cannot vouch for the authenticity of materials listed on-chain. We encourage users to be careful purchasing content and particularly careful with applications that can contain executable javascript. Any application you install *can* read your wallet. Purchase items from sellers you trust -- creator publickeys are displayed right there in the listings for this purpose.

## Decentralized Moderation

Given that the Saito Store is an index for blockchain-mediated transactions, we cannot take any stance on the reversibility of transactions. We do not condone the sale of illegal or morally odious content and will review and remove listings from our own index if they violate our private judgment. If you run into content that are clearly inappropriate and/or in violation of shared human decency, please flag it for review.
